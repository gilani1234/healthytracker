# Решение ошибки "src refspec main does not match any"

## Причина ошибки

Эта ошибка возникает, когда:
1. Репозиторий инициализирован
2. Remote настроен
3. НО: нет коммитов в репозитории

## Решение

Выполните исправленный скрипт `fix-and-push.ps1`:

```powershell
.\fix-and-push.ps1
```

## Или выполните команды вручную:

```powershell
# 1. Убедитесь, что все файлы добавлены
git add .

# 2. Создайте коммит (если его еще нет)
git commit -m "Initial commit: Calories Counter App"

# 3. Если ветка main не существует, создайте её
git branch -M main

# 4. Отправьте в GitHub
git push -u origin main
```

## Если всё ещё не работает

### Вариант 1: Принудительное создание коммита
```powershell
git commit --allow-empty -m "Initial commit"
git branch -M main
git push -u origin main
```

### Вариант 2: Если репозиторий на GitHub не пустой
Если в репозитории на GitHub уже есть коммиты (например, README был создан автоматически), выполните:

```powershell
git pull origin main --allow-unrelated-histories
git push -u origin main
```

### Вариант 3: Проверка текущей ветки
```powershell
# Проверьте, какая ветка активна
git branch

# Если видите ветку master вместо main:
git branch -M master main
git push -u origin main
```

## Проверка статуса

Для проверки текущего состояния выполните:
```powershell
git status
git log --oneline
git branch
```
